getElement(expression) → {HTMLElement|null}

PUBLIC helper function.
Determines if the passed expression is a string-id or an HTML-element. If it is an HTML-element, the method returns this element. If the parameter is a string, the method assumes that the parameter is the id of the element we're looking for. In case the expression is from another type or the passed id does not exist, the method returns null.
