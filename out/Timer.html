Class: Timer

Timer(hours, minutes, seconds, display, run)

new Timer(hours, minutes, seconds, display, run)

Creates a new timer class with following features:
METHODS: 9
PROPERTIES: 14
EVENTS: 2
- timeout --> fired when countdown reaches zero
- timerexpired --> fired when defined alert time is reached
EXAMPLE: let myTimer = new Timer (12, 30, 0, 'divClock', true);
Will assign the timer to 'myTimer' at "12:30:00" and starts it immediately.
Running time will be displayed on a <div>-Element with ID 'divClock'.
USAGE: All params while creating a new instance of the timer are optional!
If no time is submitted, the timer starts with 00:00:00.
If no HTML-element is submitted, it can be set later by the displayElement property.

Parameters:
Name Type Default Description
hours number | string 0 Either the hours to be set as a number or the whole time to be set as a valid time string like: "18:45:30"
minutes number 0 [Optional]
seconds number 0 [Optional]
display HTMLElement | string HTML element or it's id to display the time.
run boolean false Determines wether the new timer starts immediately or not (default).

countDown(seconds)

Starts or stops a countdown depending on the submitted parameter "seconds".
If seconds are decreased to zero, a timeout-event is fired and the countdown timer stops. It is possible to play a sound (i.e. a ticking clock) while the countdown runs.
