Module: js

Creates a collection to store and handle different types of data. You can work with these different data types in several ways. Each item of the collection is accessable over it's numerical index or over an assigned unique string key. You can filter and access the whole list by the different types, and add new items or change and remove existing items. Besides the common data types, you can determine wether a stored item is an array or not.
