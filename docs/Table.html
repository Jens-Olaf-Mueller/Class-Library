Class: Table

Table(table, content, headers, parent, stylesheetnullable)

new Table(table, content, headers, parent, stylesheetnullable)

Creates a new Table class and a HTML-table-element, according to the passed content. All parameters excpt from table are optional.
Parameters:
Name Type Attributes Description
table string | HTMLElement Mandatory parameter! The reference or the id of the table to be created.
content Array.<object> | Array.<string> | string The content to be displayed in the table. The parameter can be either a string, separated by the set splitter property (default is comma) or a string array or an object consisting of key-value pairs. If 'content' is an object, the keys are used as column captions (headers).
headers Array.<string> | string A string array or a connected string, splittable by the classe's splitter-property
parent string | HTMLElement The parent HTML-element, it's id or the document.body if omitted.
stylesheet string <nullable>
 The path to a related style sheet in order to style the table.
