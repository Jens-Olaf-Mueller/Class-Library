(async) show(prompt, title, buttons, modal, gradient)

The main method of the class. Creates and displays a messagebox according to the settings and the passed parameters. The calling function must be marked with the key word AWAIT.
Parameters:
Name Type Description
prompt string [Optional] The displayd message to the user. If no message is passed, a non-modal default box with a single "OK" button will be displayed.
title string [Optional] The Caption of the message box. If omitted, the page title is used.
buttons string | Array.<string> [Optional] String, separeted by comma or string array. If omitted, a single "OK" button is displayed.
modal boolean [Optional] Tells wether the messagebox is displayed modal (default) or not
gradient boolean [Optional] Tells wether the title bar has got a color gradient (default) or not.
