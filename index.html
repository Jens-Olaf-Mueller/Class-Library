The color class allows you simple color handling.

You can convert colors between RGB, HSL, HWB, CMYK format and HEX-codes.
Further you can use named colors and add new own colors. Another feature is recognizing the brighter one of two colors. This can be used to switch automatically the text color depending on the background.
